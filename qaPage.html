Question-1: What are the different ways to select an element in the DOM?

We can select elements in the DOM in five ways. Those are: 1. "getElementById()", 2. "getElementsByClassName()",
3. "getElementsByTagName()", 4. "querySelector()", 5. "querySelectorAll()"

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML : By using this we can get the inner HTML or set and HTML structure.
innerText : By using this we can get the inner text or set/change the text inside an element.
textContent : By using it we can get/set all text, including hidden elements.

Question-3 :What is event delegation in the DOM?

It is a technique in js which allow us to attach a single event listener to a parent element to handle events on its child elements.

Question-4: What is event bubbling in the DOM?

This is event which goes upwards and grabs the innermost elements and then its parents.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating an Element : We will use "document.createElement()" to create a new element.
Adding an Element : We will use "appendChild()" to add an element to the last child of the parent element.
Remove an Element : We will use "remove()" to remove an element.
